jquery.freezetable - fixed height demo

Setting 'autoHeight' to false will fix the height of the table. The 'height' option determines the height of the table body in pixels. 

$(document).ready(function() {
    $('#box-table-a').freezeTable({
        'autoHeight': false,
        'height'    : 160
    });
});
